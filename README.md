# Доля

Середовище тестування для [Мавки](https://xn--80aaf6ah.xn--j1amh/).

Проєкт знаходиться у стадії розробки.

## Встановлення

1. Встановіть залежності:

    ```bash
    npm install
    ```

2. Запустіть усі тести у директорії `тести/`:

    ```bash
    npm run test
    ```
